# Rasterizer
### 3D Scene Renderer by Rasterization using C++ and OpenGL

## INSTRUCTIONS TO RUN:
In the main folder, run the following commands in the terminal

```
mkdir build
cd build
cmake ..
make -j 8
```

This creates the executable Rast. You can render a scene by typing:

```
./Rast bunnyscene.dae
```
or 
```
./Rast staircase.dae
```
